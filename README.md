## Requirements

- [Node.js](http://nodejs.org/)
- Grunt-CLI `npm install -g grunt-cli`

## Usage

- `git clone git@github.com:Kageetai/JayMap.git`
- `npm install`
- `bower install`
- `grunt` or `grunt serve` or `grunt serve:dist`
